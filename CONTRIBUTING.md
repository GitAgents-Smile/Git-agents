# Contributing to Git Agents

Thanks for your interest in contributing! 🎉

## How to Contribute

1. **Fork** this repository
2. **Clone** your fork: `git clone https://github.com/<your-username>/Git-Agents.git`
3. **Create a branch**: `git checkout -b feat/your-feature`
4. **Make your changes** to `index.html` (or supporting files)
5. **Test locally** by opening `index.html` in a browser or running a local static server
6. **Commit** with a clear message: `git commit -m 'feat: add pricing toggle'`
7. **Push**: `git push origin feat/your-feature`
8. **Open a Pull Request** describing what you changed and why

## Guidelines

- Keep the single-file structure unless a change genuinely requires splitting CSS/JS out
- Match the existing "sticker" design language (hard shadows, thick borders, warm palette)
- Test in both desktop and mobile viewport sizes before submitting
- Avoid adding external dependencies unless necessary — this project is intentionally dependency-light

## Reporting Bugs

Please use the [Bug Report template](../../issues/new?template=bug_report.md) and include:

- Steps to reproduce
- Expected vs actual behavior
- Browser and OS

## Suggesting Features

Please use the [Feature Request template](../../issues/new?template=feature_request.md).
