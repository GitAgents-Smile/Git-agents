# Security Policy

## Supported Versions

This is a static demo project without versioned releases. Security fixes are applied to the `main` branch only.

| Version | Supported |
| ------- | --------- |
| main    | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**. Instead:

1. Open a [private security advisory](../../security/advisories/new) on GitHub, or
2. Contact the maintainer directly.

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

We aim to respond within 5 business days.

## Important Note on This Project

This repository contains a **client-side demo authentication system** (in-memory, no backend, no encryption, no persistence). It is intended purely for UI/UX demonstration purposes and **must not be used as-is in any production system** that handles real user credentials.
